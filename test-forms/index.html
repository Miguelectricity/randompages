Ghostwriter Test Forms

Test job application forms for end-to-end testing of the ghostwriter agent.

  • 1. Single-Page Form

    Standard form with all fields visible on one page. Tests basic form filling and validation.

  • 2. Apply Now → Form

    Job listing page with "Apply Now" button leading to a separate form page. Tests navigation before form.

  • 3. Multi-Page Wizard

    5-step application wizard with navigation between pages. Tests multi-step form handling.

  • 4. Conditional Fields

    Form with fields that appear/hide based on other inputs. Tests dynamic field detection.

  • 5. Dynamic JS Rendering

    Form fields rendered via JavaScript after page load. Tests waiting for dynamic content.

  • 6. Expandable Sections (Breezy HR Style)

    Work Experience and Education sections with "Add Another" functionality. Tests dynamic entry creation with multiple sub-fields per entry.

  • 7. Rippling Style Form

    Complex form with custom questions, radio groups, checkboxes, dropdowns, and EEOC section. Based on Rippling ATS.
